=== Clube Serben Connect ===
Version: 0.4.1

Integração WordPress com a API Clube Serben.

Shortcodes:
[serben_app] - Login por CPF, criação/sincronização automática de usuário WP e área do associado.
[serben_login] - Formulário de login por CPF.
[serben_logout] - Logout do associado.
[serben_lookup] - Consulta de CPF sem autenticar.
[serben_register] - Cadastro integrado via POST Clientes.

Observação: o login por CPF é uma etapa de homologação. Para produção, recomenda-se autenticação com senha, token ou OTP quando a API disponibilizar esse endpoint.
